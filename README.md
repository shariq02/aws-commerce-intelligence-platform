# AWS Commerce Intelligence Platform

A production-grade multi-domain operational intelligence platform built on AWS, Apache Flink, Databricks, and a modern open-source data stack.

## Domains
- E-Commerce Orders
- Pharmacy and Health Retail
- Marketplace Seller Operations

## Tech Stack
Flink | Kafka | Databricks | Delta Lake | dbt | AWS S3 | DynamoDB | Lambda | FastAPI | Grafana | Prefect | Terraform | GitHub Actions
